 Project: Control Led strip based on sound.   
 
 This code works in arduino UNO, using the pins:  
  
 LED STRIP:  
 
 RED - PIN (PWN) 11  
 GREEN - PIN (PWN) 10  
 BLUE - PIN (PWN) 9  
  
 SOUND SENSOR:  
   
 SENSOR - PIN (A) 2  
  
 Remember: Between LED and Arduino use a resistor 220 oms or 300 oms
